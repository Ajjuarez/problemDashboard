The application is initated by the index.ejs file which calls upon partials from the partials folder in the following order:

```
Views
│   README.txt
│   index.ejs  
│
└───partials
│   │   head.ejs
│   │   body.ejs
│   │   modals.ejs
│   │   menu.ejs
│   │   dataDisplay.ejs
│   │   footer.ejs
│
└───demo 
 
```
